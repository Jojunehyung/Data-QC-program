# Spec: Master DB 생성

## 목적
다수의 수집일지 엑셀 파일을 병합하여 통합 Master DB를 생성합니다.

## 입력
- 수집일지 엑셀 파일 1개 이상 (다중 선택 가능)
- 필수 컬럼: 내부코드, 식별번호, 병록번호, 이름, 개인번호, 접수일자

## 처리 순서
1. 파일 읽기 및 전체 시트 병합
2. 병록번호 정규화 (7자리 → 8자리)
3. 이름에서 가족 키워드 탐지 → 가족데이터 분류
4. 개인번호 파싱 → 생성ID 자동 생성
5. 식별번호 우선순위 정렬 (P > F > 기타)
6. 중복 제거 (병록번호 기준 대표 행 선택)

## 출력
`Master_DB.xlsx` — 입력 파일과 동일 폴더 저장

| 시트 | 내용 |
|---|---|
| 통합데이터 | 유효 환자 데이터 |
| 오기데이터 | 이상 데이터 |
| 가족데이터 | 가족/보호자 분류 |

## 불변 규칙
- P식별번호가 있으면 F식별번호는 반드시 가족데이터로
- 병록번호 텍스트 서식 적용 (앞자리 0 유지)
