# Spec: R-ID 매칭

## 목적
IRB·DRB 승인 하에 수령한 R-ID 파일과 Master DB를 자동 매칭합니다.

## 입력
- Master_DB.xlsx
- R-ID 파일 1개 이상 (다중 선택 가능)

## 처리 순서
1. Master DB 읽기
2. R-ID 파일 읽기 및 컬럼 자동 감지
3. 내부 식별키 기반 매핑 구축
4. 매칭 수행
5. 결과 분류: 매칭성공 / 미매칭 / 중복

## 출력
`매칭결과.xlsx`

| 시트 | R-ID | 내용 |
|---|---|---|
| 매칭성공 | ✅ | 매칭된 데이터 |
| 미매칭 | ❌ | 매칭 실패 |
| 중복 | ❌ | 내부 식별키 중복 건 |

## 불변 규칙
- 이미 R-ID가 있는 행은 재매칭하지 않고 보존
- 미매칭·중복 시트에서 R-ID 컬럼 제거
- 생성ID 컬럼은 모든 출력 시트에서 제거 (내부 처리용 키, 외부 파일에 포함 금지)
