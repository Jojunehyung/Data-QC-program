# Spec: 수집일지 자동 수정

## 목적
수정 완료된 데이터를 수령했을 때, 원본 수집일지 엑셀 파일을 자동으로 업데이트합니다.
병록번호를 기준으로 매칭하여 변경된 값만 덮어씁니다.

## 입력
- 원본 수집일지 파일 1개 이상
- 수정 소스 선택: 휴비스쌤 / 슈프림
- 수정파일

## 수정파일 형식

### 휴비스쌤
- 원본 수집일지와 동일한 컬럼 구조
- 병록번호 기준 매칭

### 슈프림
- 시트명: `슈프림 추출 데이터(프로토콜번호)` 형식
- 컬럼: 병록번호, 환자명, 성별, 생년월일
- 컬럼명 자동 감지 (병록·환자번호, 성별, 생년월일·생년 등)

## 처리 순서
1. 원본 수집일지 파일 읽기 (전체 시트)
2. 수정파일 읽기 및 병록번호 기준 매핑 구축
3. 원본 파일의 각 행에서 병록번호 매칭
4. 매칭된 행의 변경값만 덮어쓰기
5. 원본과 동일한 시트 구조로 출력

## 출력
`{원본파일명}_수정완료.xlsx` — 원본 파일과 동일 폴더 저장
- 시트 구조 및 컬럼 순서 원본과 동일
- 병록번호 텍스트 서식 유지

## 완료 메시지 필수 포함 항목
- 처리 파일 수
- 수정된 셀 수
- 저장 파일명
