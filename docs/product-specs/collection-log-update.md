# Spec: 수집일지 자동 수정

## 목적
수정 완료된 데이터를 수령했을 때, 원본 수집일지 엑셀 파일을 자동으로 업데이트합니다.
휴비스쌤과 슈프림은 매칭 키와 업데이트 방식이 다릅니다.

## 입력
- 원본 수집일지 파일 1개 이상
- 수정 소스 선택: 휴비스쌤 / 슈프림
- 수정파일

## 수정파일 형식 및 매칭 방식

### 휴비스쌤
- 형식: `bCODE | 성별 | 출생연도 | 출생월` (첫 번째 데이터 행은 내부 필드명 행 → 자동 skip)
- 매칭 키: **bCODE** (병록번호 없음)
- 업데이트 방식: 성별·출생연도·출생월 → 개인번호(예: F93.05) 재조합 후 수집일지에 반영

### 슈프림
- 형식: 시트명 `슈프림 추출 데이터(프로토콜번호)`, 컬럼명 자동 감지
- 매칭 키: **병록번호**
- 업데이트 방식: 변경된 컬럼(성별·생년월일·이름 등)을 직접 덮어쓰기

## 처리 순서
1. 원본 수집일지 파일 읽기 (전체 시트)
2. 수정파일 읽기 및 매핑 구축
   - 휴비스쌤: bCODE → (성별, 출생연도, 출생월) 매핑
   - 슈프림: 병록번호 → 수정 행 매핑
3. 원본 파일 각 행에서 키 매칭
4. 매칭된 행 업데이트
   - 휴비스쌤: 개인번호 재조합 (`_build_pers_no`)
   - 슈프림: 변경된 컬럼 값 덮어쓰기
5. 원본과 동일한 시트 구조로 출력

## 출력
`{원본파일명}_수정완료.xlsx` — 원본 파일과 동일 폴더 저장
- 시트 구조 및 컬럼 순서 원본과 동일
- 병록번호 텍스트 서식 유지

## 완료 메시지 필수 포함 항목
- 처리 파일 수
- 수정된 셀 수
- 저장 파일명
