# Spec: 오류 알림 발송

## 목적
미매칭 데이터 목록을 담당자에게 이메일로 알립니다.
데모 환경에서는 실제 발송 없이 미리보기만 표시합니다.

## 입력
- 매칭결과.xlsx (미매칭 시트)
- 발송 대상 시스템 선택: 수집관리시스템 / 데이터추출시스템
- 수신자 이메일 주소

## 처리 순서
1. 매칭결과 파일에서 미매칭 시트 읽기
2. 발송 대상 선택 (수집관리시스템: 내부코드 목록 / 데이터추출시스템: 병록번호 목록)
3. 수신자 이메일 입력
4. 이메일 미리보기 창 표시
5. 발송 확인 시 완료 메시지 (데모: 실제 발송 없음)

## 발송 내용

| 항목 | 내용 |
|---|---|
| 제목 | `[데이터 오류 알림] {시스템} 미매칭 건 확인 요청 ({날짜})` |
| 본문 | 미매칭 ID 목록 (최대 50건 표시, 초과 시 "외 N건") |

## 데모 원칙
- 실제 SMTP 연결 없음
- 미리보기 창에서 발송 내용 확인만 가능
- 실제 환경 연결은 주석(`# smtp.company.com:587`)으로만 표시
