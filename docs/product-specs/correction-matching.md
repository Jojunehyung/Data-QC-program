# Spec: 오기입 보정 매칭

## 목적
외부 데이터소스(Hubis) 기준으로 성별·생년월일 오기입을 보정한 후 재매칭합니다.

## 입력
- 매칭결과.xlsx (미매칭 시트)
- 외부 데이터소스(Hubis) 파일: bCODE, 성별, 출생연도, 출생월
- R-ID 파일 1개 이상

## 처리 순서
1. 미매칭 데이터 읽기
2. Hubis 기준으로 성별·생년월 보정
3. 보정된 값으로 내부 식별키 재생성
4. R-ID 파일과 재매칭

## 출력
`보정매칭결과.xlsx`

| 시트 | R-ID | 내용 |
|---|---|---|
| 매칭성공 | ✅ | 재매칭 성공 |
| 미매칭 | ❌ | 보정 후에도 실패 |
| 중복 | ❌ | 내부 식별키 중복 건 |

## 완료 메시지 필수 포함 항목
- 오기입 보정 건수
- 재매칭 성공 건수
- 미매칭 건수
- 중복 건수
