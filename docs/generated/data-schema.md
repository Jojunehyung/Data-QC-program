# Data Schema — 자동 생성 문서

## 컬럼 상수 목록

| 상수 | 값 | 타입 |
|---|---|---|
| COL_BCODE | 내부코드 | 숫자 문자열 |
| COL_ID_NO | 식별번호 | 문자열 |
| COL_HOSP_NUM | 병록번호 | 8자리 문자열 |
| COL_NAME | 이름 | 한글 문자열 |
| COL_PERS_NO | 개인번호 | M/F + YY.MM |
| COL_DATE | 접수일자 | date |
| COL_GEN_ID | 생성ID | 문자열 |
| COL_RID | R-ID | 문자열 |
| COL_GENDER | 성별 | M / F |
| COL_BIRTH | 생년월일 | 문자열 |
| COL_NOTE | 비고 | 문자열 |
