# Reference: 컬럼 상수 & 키워드

| 상수 | 값 | 용도 |
|---|---|---|
| COL_BCODE | bCODE | 기관 환자 코드 |
| COL_ID_NO | 식별번호 | O/X 식별자 |
| COL_HOSP_NUM | 병록번호 | 입원 번호 |
| COL_NAME | 이름 | 환자명 |
| COL_PERS_NO | 개인번호 | 성별+출생연월 |
| COL_DATE | 접수일자 | 수집일 |
| COL_GEN_ID | 생성ID | 내부 식별키 |
| COL_RID | R-ID | 연구 ID |
| COL_GENDER | 성별 | M / F |
| COL_BIRTH | 생년월일 | 생년월일 |
| COL_NOTE | 비고 | 메모 |
