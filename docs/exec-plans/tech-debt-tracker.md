# Tech Debt Tracker

## 현재 기술 부채

| 항목 | 내용 | 우선순위 |
|---|---|---|
| `YEAR_THRESHOLD` 하드코딩 | 현재 24로 고정, 동적 연월 기준으로 전환 필요 | 높음 |
| 이메일 발송 미구현 | 데모용 미리보기만 제공, 실제 SMTP 연결 없음 | 중간 |
| 수집일지 자동 수정 — 컬럼 충돌 처리 미흡 | 원본·수정파일 컬럼명 불일치 시 silent skip | 낮음 |

## 향후 개선 항목

| 항목 | 내용 |
|---|---|
| SMTP 연결 | 실제 환경에서 회사 메일 서버 연동 |
| 로그 파일 출력 | 콘솔 print를 파일 로그로 전환 |
| 수정 전·후 diff 뷰어 | 수집일지 수정 시 변경 내용 비교 창 |
